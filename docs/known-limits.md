# Known Limits

This package is intentionally narrow.

- It is a local bridge to Robinhood's official Agentic Trading MCP, not a brokerage platform.
- It does not provide investment advice, strategy execution, portfolio optimization, or tax handling.
- It does not use private Robinhood web or mobile APIs.
- It does not manage production hosting, TLS termination, or public-network access.
- It does not guarantee Robinhood MCP tool availability, schemas, or account eligibility.
- It does not bypass Robinhood account, Agentic Trading, market, or compliance restrictions.

If a feature does not help setup, local MCP forwarding, safety controls, or the thin SDK, it is probably out of scope.

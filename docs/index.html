Docker-first local MCP bridge

Connect MCP clients to Robinhood Agentic Trading in one command.

A lean local bridge, Python SDK, and safety layer for Robinhood's official Trading MCP. No private Robinhood web or mobile APIs.

01

Quickstart

Clone, authenticate, start the bridge, and check it.

1

Clone

git clone https://github.com/rithwikbabu/Robinhood-ATS.git
cd Robinhood-ATS
2

Bootstrap

make bootstrap

This creates `.env`, creates `.rh-mcp-state/`, opens browser OAuth, starts Docker Compose, and runs readiness checks.

3

Use the URL

make url

Default: http://127.0.0.1:8080/mcp

02

Connect an MCP client

Use Streamable HTTP. Keep the bridge local unless you add transport security.

Basic config

{
  "mcpServers": {
    "robinhood": {
      "type": "streamable-http",
      "url": "http://127.0.0.1:8080/mcp"
    }
  }
}

With local bearer token

{
  "mcpServers": {
    "robinhood": {
      "type": "streamable-http",
      "url": "http://127.0.0.1:8080/mcp",
      "headers": {
        "Authorization": "Bearer ${LOCAL_MCP_BEARER_TOKEN}"
      }
    }
  }
}
03

Python SDK

Install the package when you want a small Python client or direct CLI usage. Docker remains the recommended bridge runtime. 

pip install robinhood-mcp-bridge
from robinhood_mcp_bridge import RobinhoodMCPClient

client = RobinhoodMCPClient()
accounts = client.get_accounts()
quotes = client.get_equity_quotes(symbols=["AAPL"])
04

Safety model

The default posture is read-oriented and local-first.

No private APIs

Forwards to Robinhood's official Agentic Trading MCP endpoint only.

Loopback by default

Docker publishes the local bridge on host loopback, not a public interface.

Order guardrails

`place_equity_order` requires live trading enabled, dry-run disabled, and a matching recent `review_equity_order`.

Audit trail

Local state and audit logs live in `.rh-mcp-state/` for Docker usage.

05

Configuration

Most users only change ports or trading guardrails.

Variable Default Use
MCP_HOST_PORT 8080 Host port for the local MCP bridge.
AUTH_HOST_PORT 8765 Temporary OAuth callback port during login.
LOCAL_MCP_BEARER_TOKEN unset Optional bearer token for local MCP clients.
DRY_RUN true Blocks forwarding real equity placement.
LIVE_TRADING false Must be true before real equity orders can forward.
SYMBOL_ALLOWLIST unset Optional comma-separated tradable symbols.

Full reference: configuration.md

07

Troubleshooting

Start with the checks. They are designed to be fast.

Auth missing

make bootstrap

Port busy

MCP_HOST_PORT=8081 make start
AUTH_HOST_PORT=8766 make auth

Need a clean login

make reset-state
make bootstrap

Inspect service

make ps
make logs
